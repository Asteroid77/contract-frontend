# Support

## Use This File For

- 说明如何获取前端仓库帮助
- 区分功能、文档和安全问题的入口

## Support Paths

- 一般开发问题：先阅读 `README.md`、`AGENTS.md`、`docs/index.md`
- 模块边界与文档迁移问题：查看 `docs/index.md` 与治理说明
- 构建、CI、脚本问题：查看 `package.json`、`scripts/`、`.github/workflows/`
- 安全问题：不要公开提问，改走 `SECURITY.md`

## What To Include

求助时请附带：

- 目标页面或模块
- 失败现象
- 已执行命令
- 控制台报错、测试输出或截图说明

## Out Of Scope

- 与前端仓库无关的后端实现问题
- 没有最小复现信息的“页面异常”描述
